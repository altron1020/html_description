NodeMCU ESP8266 V3

 

Le NodeMCU est un kit de développement open source disposant d’un SoC Wi-Fi ESP8266 fabriqué par Espressif Systems idéal pour le développement d’objets connectés (IoT).

La carte dispose également de nombreux PINs et d’un port microUSB, permettant de brancher la carte à un ordinateur (avec un câble microUSB, non inclus) pour la programmer comme une Arduino, via l’IDE. Cette dernière peut aussi se programmer avec des scripts LUA, en C++, en JavaScript, en MicroPython et en C.

Grâce à l’interface Wi-Fi, la carte permet par exemple la connexion à internet, l’hébergement d’un serveur, le partage de données, etc.

Caractéristiques et détails techniques

  • Alimentation : 

    • 5v via microUSB
    • 3.3v à 9v via le port Vin
    • 3.3v via les bornes 3.3v  (voir schéma)

 

  • Tension de fonctionnement : 3.3v

  • Consommation : 10uA~170mA

  • Microcontrôleur : ESP8266

  • MicroProcesseur : Tensilica L106 32-bit

  • Fréquence : 30 MHz

  • Mémoire RAM : 32K + 80K

  • Mémoire Flash : 16MB

  • GPIOs : 17

  • Interface Wifi : 802.11 b/g/n 2,4 GHz

  • Interfaces : I2C, SPI, UART

  • Antenne intégrée
